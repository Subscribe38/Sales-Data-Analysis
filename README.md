# Sales Data Analysis (Excel)

## Objective
To analyze sales performance across regions and product categories using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Analysis Performed
- Total sales and profit by region
- Profit analysis by product category
- Visualization using column and bar charts

## Key Insights
- West region shows the highest sales and profit
- Technology category contributes maximum profit
- Office supplies have lower profit margins

## Conclusion
This project demonstrates basic data analysis skills using Excel for business decision making.
